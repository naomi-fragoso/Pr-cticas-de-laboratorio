Concepto

También diseño responsive, es un tipo de desarrollo de páginas web que permite su óptima visualización en distintos tipos de dispositivos finales. El término diseño responsivo es tan antiguo como la web móvil y se generalizó en torno a 2013.

Características de Diseño Responsivo

1. Prefieren los formatos verticales

2. Funciona mediante la priorización de elementos

3. Implementa funcionalidades únicas

4. Se basan en la interacción con el usuario

5. Automatiza labores de diseño y programación

Elementos

*Tamaño de visualización

*Tamaño de navegación

*Contenidos textuales

*Contenidos visuales

*Botones de acción

Características

Ejemplos

*Google

*Video juegos

*Redes Sociales

Importancia

Hoy día, más de la mitad de los accesos a sitios web en todo el mundo se realizan desde dispositivos móviles. Este dato demuestra la importancia de que las empresas ofrezcan una experiencia de navegación óptima en estas plataformas. Esta necesidad también se aplica a sitios web más pequeños, como los micrositios web o los sitios web one page.
